    前序准备

    • 你的本地环境需要安装 node 和 git。我们的技术栈基于 ES2015+、React、dva、g2 和 antd，提前了解和学习这些知识会非常有帮助。

    安装

    有两种方式进行安装：

    直接 clone git 仓库# $ git clone --depth=1 https://github.com/ant-design/ant-design-pro.git my-project $ cd my-project 使用命令行工具# 你可以使用集成化的命令行工具 ant-design-pro-cli。

    $ npm install ant-design-pro-cli -g $ mkdir my-project && cd my-project $ pro new # 安装脚手架
